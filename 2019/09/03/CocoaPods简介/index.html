一、简介

CocoaPods 是开发 OS X 和 iOS 应用程序的一个第三方库的依赖管理工具。利用 CocoaPods，可以定义自己的依赖关系 (称作 pods)，并且随着时间的变化，以及在整个开发环境中对第三方库的版本管理非常方便。

在项目中的引入的依赖都会记录在Podfile文件中，CocoaPods会解析这些库的依赖并且自动下载源码，然后把它一起打包链接到Xcode中来构建你的项目。

  • 在工程中引入第三方代码会涉及到许多内容。CocoaPods 简化了这一切，它能够自动配置编译选项。
  • 通过 CocoaPods，可以很方便的查找到新的第三方库。

二、核心组件

1.CocoaPods/CocoaPod

每当执行一个 pod 命令时，这个组件都将被激活。该组件包括了所有使用 CocoaPods 涉及到的功能

2.CocoaPods/Core

Core 组件提供支持与 CocoaPods 相关文件的处理，文件主要是 Podfile 和 podspecs。

  • Podfile.lock：记录了已经安装的pod和其版本的文件

  • .podspec：记录当前仓库依赖的组件

3.CocoaPods/Xcodeproj

负责所有工程文件的整合。它能够对创建并修改 .xcodeproj 和 .xcworkspace 文件。

三、CocoaPods的使用

1.pod install和pod update的区别

  • 使用pod install来安装新的pod。即在Podfile中添加或删除了pod时使用
  • 使用pod update来更新你的已有的pod到一个更新的版本。
